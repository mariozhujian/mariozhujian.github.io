PicGo + Github + Typora 搭建个人图床

发表于 2024-04-23 分类于 bolg 阅读次数：

PicGo + Github + Typora 搭建个人图床

Github创建图床仓库

创建 Github 个人账户 Token

  1. 个人头像 -> Settings

  2. Developer settings

  1. Personal access settings -> Tokens(classic)

  1. Generate new token

    1. 填入 token 名称，并投选 repo，最后点击生成 token，因为新生成的 token 只展示一次，可以将其拷贝到文件备份

下载并配置 PicGo

  1. PicGo 官网下载地址：https://github.com/Molunerfinn/PicGo/releases

  2. 配置 PicGo

配置 Typora

  1. 打开 Typora 打开左上角文件并点击偏好设置，选择图像

  2. 配置完成后，点击验证图片上传选项，出现验证成功即代表图片上传成功

  3. 如果出现在 Typora 插入图片能上传到 github 成功，但图片加载不出来，可以配置本地 hosts 文件

​

​ 大功告成，可以愉快的玩耍了！！！
